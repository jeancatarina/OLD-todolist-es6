# Requirements

 - NodeJS (https://nodejs.org/en/)
 
 - Git (https://git-scm.com/)
 
# Running

choose a directory, 

- $ git clone https://github.com/jeancatarina/todolist-es6

- $ cd todolist-es6

- $ npm install

- $ npm run build
